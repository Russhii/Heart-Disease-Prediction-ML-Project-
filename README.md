# ❤️ Heart Disease Prediction System

A Machine Learning web app that predicts the likelihood of heart disease based on patient clinical data, built with **Streamlit** and **Logistic Regression**.

🔗 **Live Demo:** [Add your Streamlit app URL here after deployment]

## Features
- Interactive UI for entering patient details (age, cholesterol, blood pressure, ECG, etc.)
- Predicts risk of heart disease using a trained Logistic Regression model
- Displays prediction probability with a visual risk indicator

## Tech Stack
- Python
- Streamlit
- Scikit-Learn
- Pandas
- Joblib

## How to Run Locally
```bash
git clone https://github.com/Russhii/Heart-Disease-Prediction-ML-Project-.git
cd Heart-Disease-Prediction-ML-Project-
pip install -r requirements.txt
streamlit run app.py
```

## Model
The model is a Logistic Regression classifier trained on clinical features including age, chest pain type, resting blood pressure, cholesterol, fasting blood sugar, resting ECG, max heart rate, exercise-induced angina, oldpeak, and ST slope.
